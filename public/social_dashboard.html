Setup Guide

Step 1 — Add Render Environment Variables
1

Meta App (Instagram + Facebook)

Go to developers.facebook.com → My Apps → Create App → Business.
Add META_APP_ID and META_APP_SECRET to Render env vars.
Set OAuth redirect URI to: https://swarm-app-3nch.onrender.com/api/social/callback/meta

2

TikTok Business App

Go to developers.tiktok.com → Manage Apps → Create App.
Add TIKTOK_CLIENT_KEY and TIKTOK_CLIENT_SECRET to Render env vars.
Set redirect URI to: https://swarm-app-3nch.onrender.com/api/social/callback/tiktok

3

Connect Meta (Instagram + Facebook together)

Click "Connect with Meta" above. Log in with the Facebook account that manages Houseofjreym page and @houseofjreym Instagram. Both platforms connect in one OAuth flow.

4

Connect TikTok

Click "Connect TikTok" above. Log in as @houseofjreym. Grant video upload and insights permissions.

Manual Token Entry (alternative)

If OAuth isn't available yet, paste tokens directly from each platform's developer portal.

Token Sources:
Instagram/Facebook: Graph API Explorer at developers.facebook.com/tools/explorer

TikTok: developers.tiktok.com → Auth code tool

Tokens are stored encrypted in Supabase. Never in logs or URLs.
