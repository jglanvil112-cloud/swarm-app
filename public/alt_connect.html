📸 Instagram + Facebook NOT CONNECTED

Both platforms connect with one Meta token. Two ways to get it:

METHOD 1 — FACEBOOK BUSINESS SUITE (NO DEVELOPER ACCOUNT NEEDED)

  1. Go to business.facebook.com/settings/system-users
  2. Click Add → create a System User named "IBRAHIM"
  3. Set role to Admin → click Generate New Token
  4. Select your Houseofjreym page
  5. Check: pages_show_list pages_manage_posts instagram_basic instagram_content_publish instagram_manage_insights read_insights
  6. Copy the token → paste below
OR

METHOD 2 — META FOR DEVELOPERS (REQUIRES DEVELOPER ACCOUNT)

  1. Go to developers.facebook.com/tools/explorer
  2. Click Register if prompted — complete phone verification
  3. In the Explorer: Meta App dropdown → Create New App → type "houseofjreym"
  4. Click Generate Access Token → log in as Houseofjreym admin
  5. Check ALL permissions listed above → copy the token
OR

METHOD 3 — OAUTH BUTTON (IF APP IS CONFIGURED IN RENDER)

📸 Connect Instagram (OAuth)🔗 Connect Facebook (OAuth)
